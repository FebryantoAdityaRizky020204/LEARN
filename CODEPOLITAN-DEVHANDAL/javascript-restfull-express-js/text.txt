Perbedaan GET dan POST
GET
    -> digunakan untuk menarik/menerima data
    -> data dikirim melalui query string
    -> informasi yang dikirm terlihat di URL
    -> Jumlah data yang dikirim terbatas

POST
    -> digunakan untuk mengirim data ke server
    -> digunakan pada saat membuat/mengubah data
    -> data dikirim melalui body bukan head
    dapat mengirim data dengan tipe JSON


Sedikit Teori Tentang REST
 -> Representational State Transfer
    * sebuah arsitektur sistem terdistribusi, sederhananya adalah panduan
      komunikasi data untuk CRUD antara client dan server
    * secara konsep REST menjembatani antara data yang ada di server supaya bisa
      diproses CRUD oleh banyak sumber melalui format URL dan HTTP Verb
